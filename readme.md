# Mario Ciari

Ce site est un site destiné aux peintres

## Environnement de développement

### Pré-requis

- PHP 7.4
- Composer
- Symfony CLI
- Docker
- Docker-compose

Vous pouvez vérifier les pré-requis grâce à cette commande

```bash
symfony check:requirements
```

#### Lancer l'environnement de dev

```bash
docker-compose up -d
symfony serve -d
```
